Proyecto: Mistura

Creador: Berrospi Valladares Pool Guillermo


Descripción del proyecto:

Utilizando la tecnología Ruby y sus gemas, este proyecto permite localizar los restaurantes que pertenecen a Mistura en el mapa de google, así mismo se permitira registrar uno a uno los restaurantes, asociando su Nombre, su dirección y las coordenadas previamente ubicadas (Por fines academicos).


Arquitectura

Las gemas utilizadas para el proyecto, son:

* Gmap.- Permitira mostrar dentro del mapa los restaurantes registrados.
